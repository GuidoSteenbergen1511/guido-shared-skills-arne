# Prompt Writing Guide

## The Photography Prompt Template

Write prompts using this structure:

```
[Subject description - detailed, specific]. [Clothing/brand details if people]. [Action/pose/state]. [Background description]. The photo style is [Editorial/Commercial/Documentary], taken as a [shot size: wide/medium/close-up] shot with a [camera model] camera and a [focal length]mm lens focal length. The aperture is set to f/[value], the shutter speed is [value], and the ISO is [value]. The white balance is set to [Daylight/Cloudy/Tungsten/Auto]. The focus is [focus description with bokeh treatment]. The colour scheme is [palette name], with a [tone adjective] tone. Lighting is [specific lighting setup]. The scene is [Indoor/Outdoor location], and the subject is [subject type]. The composition follows [technique] technique, taken from [angle]. The image has been post-processed with a [color grade name] — [grade description].
```

## Prompt Rules

1. **Be hyper-specific:** camera model, lens, aperture, shutter speed, ISO, white balance
2. **Describe lighting direction and quality,** not just "good lighting"
3. **Name the color grade** and describe what it does to shadows/highlights
4. **Specify composition technique:** Rule of Thirds, Leading Lines, Symmetry, Diagonal, Triangular, Central Grouping
5. **Describe the bokeh/focus fall-off explicitly**
6. **Include the emotional tone** the image should convey
7. **Reference client-specific elements:** brand colors, products, environments

## Camera Settings Reference

### For sharp environmental shots (offices, workshops, wide scenes):
- Camera: Sony A7R IV
- Lens: 24-70mm (use 35mm for wide, 50mm for standard)
- Aperture: f/4 - f/8
- Shutter: 1/125
- ISO: 200-400
- Focus: deep, most of scene sharp

### For portrait-style (people, close-ups, emotional):
- Camera: Canon EOS R5
- Lens: 85mm or 135mm
- Aperture: f/1.8 - f/2.8
- Shutter: 1/250
- ISO: 100-200
- Focus: subject sharp, background beautifully blurred (creamy bokeh)

### For dramatic / cinematic:
- Camera: RED Komodo
- Lens: 50mm anamorphic
- Aperture: f/2.0
- Shutter: 1/48 (cinematic motion)
- ISO: 800
- Focus: shallow, with lens flare and atmospheric haze

### For product / detail shots:
- Camera: Hasselblad X2D
- Lens: 90mm macro
- Aperture: f/5.6
- Shutter: 1/200
- ISO: 100
- Focus: razor-sharp on product, subtle gradient fall-off

### For editorial flat-lay / workspace shots:
- Camera: Hasselblad X2D
- Lens: 80mm for overhead flat-lay, 55mm for angled desk scene
- Aperture: f/4 - f/5.6
- Shutter: 1/125
- ISO: 100-200
- Focus: most of scene sharp, subtle fall-off at edges

## Composition Techniques

| Technique | When to use |
|---|---|
| Rule of Thirds | Default — subject at intersection points |
| Leading Lines | Drawing eye toward key element |
| Symmetry | Authority, stability, corporate power |
| Diagonal | Energy, movement, progress |
| Triangular | Groups of people, stability with dynamism |
| Central Grouping | Product shots, statement pieces |
| Negative Space | Leaving room for text overlay on slide |

## Color Grading Presets

| Grade | Description | Best for |
|---|---|---|
| Corporate Warm | Lifted shadows, warm midtones, clean highlights | Professional people shots |
| Cinematic Teal-Orange | Cool shadows, warm skin tones, high contrast | Dramatic impact slides |
| Clean Editorial | Neutral tones, slight desaturation, bright | Product, workspace shots |
| Golden Hour | Warm throughout, soft contrast, amber highlights | Outdoor, aspirational |
| Tech Modern | Cool blue tones, high clarity, sharp | Innovation, technology |
| Warm Editorial | Cream-toned lifted shadows, warm orange midtones, soft peach highlights, desaturated non-warm tones | Default style workspace/editorial shots |

## Common Mistakes to Avoid

- Too many elements in the prompt (keep it focused on one scene)
- Forgetting to specify the background (model fills with generic)
- Using generic lighting ("natural light" is too vague — say "soft window light from camera-left, creating gentle shadows on the right side of the face")
- Not specifying what the person is DOING (static poses look lifeless)
- Conflicting style cues (cinematic + corporate + editorial in one prompt)
