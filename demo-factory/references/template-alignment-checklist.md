# Template Alignment Checklist

Use this for template-driven demos (IC memo, underwriting Excel, legal standards).

1) Section Mapping
- [ ] Extract template section headers from source template
- [ ] Map each section to a dedicated extractor and writer (for swarms)
- [ ] Confirm section order matches template

2) Label Fidelity
- [ ] Preserve exact labels from template (headers, table labels)
- [ ] Preserve order and grouping of labels

3) Layout Fidelity
- [ ] Match table structure and column order
- [ ] Preserve key callouts (currency notes, totals)
- [ ] Match spacing and section breaks

4) Evidence Mapping
- [ ] Each section has explicit evidence sources
- [ ] Conflicts logged with resolution rules

5) Output Check
- [ ] Output uses the same section order
- [ ] Missing items flagged as PLACEHOLDER
