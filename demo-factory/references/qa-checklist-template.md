# Prompt QA Checklist (Template)

## 1) Objective and Role
- [ ] Role is explicit and specific
- [ ] Objective is outcome-oriented and clear

## 2) Inputs and Constraints
- [ ] Inputs are listed and concrete
- [ ] Input docs are inputs only (no output examples)
- [ ] Synthetic inputs approved by user (if used)
- [ ] "No invention" rule is present
- [ ] Placeholder rule is defined

## 3) Output Format
- [ ] Output structure is explicit
- [ ] Ordering matches reference template

## 4) Source Handling
- [ ] Fact Register or Source Log required
- [ ] No inline citations unless requested

## 5) Template Alignment
- [ ] Labels and ordering match input template
- [ ] Style/format fidelity confirmed

## 6) Completeness
- [ ] Edge cases covered (missing data, conflicts)
- [ ] Final output contains all required sections

Result:
- [ ] PASS
- [ ] NEEDS FIX (list gaps)
